# AIPICK 서비스 기획 — 제로베이스

> **핵심 정의**: "AI를 모르는 사람도, 레시피만 따라하면 결과물을 뽑아낼 수 있는 곳"

---

## 핵심 문제 정의

| 문제 | 구체적 상황 |
|------|------------|
| **발견의 벽** | "PPT 만들 때 쓸 AI가 있다는데 뭔지 모르겠다" |
| **선택의 벽** | "비슷한 이미지 생성 AI가 10개인데 뭘 써야 하지?" |
| **사용의 벽** | "ChatGPT 깔았는데 뭘 어떻게 물어봐야 하지?" |
| **조합의 벽** | "영상 만들려면 AI를 3개 써야 한다는데 순서가?" |
| **비용의 벽** | "무료로 어디까지 할 수 있지? 갑자기 결제됐다" |

---

## 타겟 사용자 페르소나

| 페르소나 | 설명 | 핵심 니즈 |
|---------|------|----------|
| **AI 입문자 (민지)** | 대학생, AI 써보고 싶은데 어디서 시작할지 모름 | "나한테 맞는 첫 번째 AI를 알려줘" |
| **목적 지향 사용자 (상훈)** | 직장인, 보고서/PPT 만들어야 하는데 AI로 빠르게 | "이 작업을 AI로 끝내는 가장 빠른 방법" |
| **AI 탐험가 (수연)** | 프리랜서, 여러 AI를 조합해서 워크플로우 구축 | "새로운 AI 도구 + 조합법을 계속 알고 싶다" |
| **교육자/학부모 (정민)** | 자녀/학생에게 안전한 AI 도구를 추천하고 싶음 | "아이가 써도 안전하고 학습에 도움되는 AI" |

## 핵심 기능 설계 (5개 축)

### 축 1. 목적 기반 탐색 — "뭘 하고 싶으세요?"

기존 AI 디렉토리들은 **도구 이름 중심**으로 나열한다. 사용자는 도구 이름을 모른다. **목적/작업 중심**으로 전환해야 한다.

```
[사용자 입력]
"유튜브 썸네일 만들고 싶어"

[AIPICK 응답]
┌─────────────────────────────────────────────┐
│ 🎯 유튜브 썸네일 만들기                        │
│                                              │
│ 추천 방법 3가지:                               │
│                                              │
│ ① 가장 쉬운 방법 (초보자용)                     │
│   → Canva AI (무료 5회/월)                     │
│   → 난이도: ★☆☆  예상 시간: 3분                │
│                                              │
│ ② 가장 퀄리티 높은 방법                         │
│   → Midjourney + Photopea                    │
│   → 난이도: ★★☆  예상 시간: 10분               │
│                                              │
│ ③ 완전 무료 방법                               │
│   → Leonardo AI + Remove.bg                  │
│   → 난이도: ★★☆  예상 시간: 7분                │
│                                              │
│ [레시피 보기] [바로 시작하기]                     │
└─────────────────────────────────────────────┘
```

**세부 설계:**

- **자연어 검색**: "보고서 요약해줘", "로고 만들어줘" 같은 일상 언어로 검색
- **목적 트리**: 대분류(이미지/텍스트/영상/음악/코드/데이터) → 중분류(생성/편집/분석/변환) → 소분류(배경제거/스타일변환/해상도업스케일)
- **상황 태그**: #무료 #초보자 #한국어지원 #모바일가능 #회원가입불필요
- **비교 테이블**: 같은 목적의 도구 3~5개를 가격/품질/속도/난이도로 비교

### 축 2. AI 레시피 — "따라만 하세요"

**이것이 AIPICK의 핵심 차별화 기능이다.**

사용자가 원하는 결과물을 얻기 위해 여러 AI 도구를 **어떤 순서로, 어떤 설정으로, 어떤 프롬프트로** 사용해야 하는지를 요리 레시피처럼 제공한다.

**레시피 핵심 요소:**

| 요소 | 설명 |
|------|------|
| **준비물** | 필요한 AI 도구 목록 + 계정 종류(무료/유료) + 사전 준비 파일 |
| **단계별 가이드** | 스크린샷/GIF 포함, 클릭 순서까지 안내 |
| **복사 가능한 프롬프트** | 한 번 클릭으로 복사, [빈칸]만 채우면 됨 |
| **설정값 명시** | 모델, 크기, 스타일 등 정확한 세팅 |
| **대안 도구** | 각 단계에서 대체 가능한 도구 표시 |
| **예상 시간 + 비용** | 스텝별/전체 시간, 무료 가능 여부 |
| **완성 예시** | 실제 결과물 이미지로 기대치 설정 |
| **난이도 분기** | 같은 목적 → 초보용/중급용/고급용 레시피 분기 |

**레시피 카테고리 예시:**

```
📁 콘텐츠 제작
  ├── 유튜브 썸네일 만들기
  ├── 인스타그램 릴스 만들기
  ├── 블로그 포스팅 자동화
  └── 팟캐스트 → 블로그 변환

📁 업무 생산성
  ├── 회의록 → 요약 → 액션아이템 추출
  ├── 경쟁사 분석 보고서 만들기
  ├── 이메일 자동 답장 템플릿
  └── 엑셀 데이터 → 인사이트 리포트

📁 학습/교육
  ├── 논문 읽고 핵심 정리하기
  ├── 외국어 단어장 자동 생성
  ├── 수학 문제 풀이 과정 이해하기
  └── 면접 준비 시뮬레이션

📁 크리에이티브
  ├── AI 음악 만들기 (작곡→편곡→마스터링)
  ├── 로고 디자인 (아이디어→생성→벡터화)
  ├── 웹툰 캐릭터 만들기
  └── 제품 목업 사진 만들기

📁 개발
  ├── 아이디어 → MVP 웹사이트 하루 만에 만들기
  ├── 버그 디버깅 자동화 워크플로우
  ├── API 문서 자동 생성
  └── 코드 리뷰 자동화 파이프라인
```

### 축 3. 스마트 추천 엔진 — "당신에게 딱 맞는 AI"

**온보딩 퀴즈 (3단계, 30초)**

```
Q1. 주로 어떤 작업을 하시나요? (복수 선택)
  □ 글쓰기/문서 작업    □ 이미지/디자인
  □ 영상 편집          □ 코딩/개발
  □ 데이터 분석        □ 학습/공부
  □ 마케팅/SNS        □ 음악/오디오

Q2. AI 사용 경험은?
  ○ 처음이에요
  ○ ChatGPT 정도 써봤어요
  ○ 여러 AI를 사용하고 있어요

Q3. 예산은?
  ○ 무료만
  ○ 월 1만원 이내
  ○ 월 5만원 이내
  ○ 비용 상관없음
```

### 축 4. 무료 사용량(Quota) 인텔리전스

사용자가 가장 민감한 정보 = **"이거 무료로 얼마나 쓸 수 있어?"**

```
┌─────────────────────────────────────────────┐
│ 💰 ChatGPT 무료 사용 정보                     │
│                                              │
│ ✅ 무료로 할 수 있는 것                        │
│  · GPT-4o mini: 무제한                       │
│  · GPT-4o: 하루 ~16회                        │
│  · 이미지 생성 (DALL-E): 하루 2회              │
│  · 파일 업로드 분석: 가능                      │
│                                              │
│ ❌ 유료 전용 기능                              │
│  · GPT-4o 무제한                             │
│  · 고급 데이터 분석                            │
│  · 맞춤 GPT 생성                              │
│                                              │
│ 💡 꿀팁                                      │
│  · 무료 한도 초과 시 → Gemini로 대체 가능       │
│  · 이미지 생성 한도 초과 시 → Leonardo AI 추천  │
│                                              │
│ 📅 리셋: 매일 자정 (한국 시간)                  │
│ 🔄 최종 업데이트: 2026.02.25                   │
└─────────────────────────────────────────────┘
```

**핵심 기능:**
- **무료 사용량 데이터베이스**: 모든 주요 AI 도구의 무료 티어 정보를 수집/업데이트
- **한도 초과 시 대안 추천**: "X가 소진됐으면 Y를 써보세요"
- **무료 조합 최적화**: "이 3개를 조합하면 유료 구독 없이도 가능"
- **가격 비교표**: 같은 기능의 유료 플랜을 나란히 비교
- **가격 변동 알림**: 무료 정책 변경 시 즉시 알림

- **프롬프트 공유**: 복사 버튼 + 실제 결과물 이미지 필수 첨부
- **Before/After**: AI 적용 전후 비교
- **레시피 ↔ 커뮤니티 연동**: 레시피 사용 후 결과물 공유 → 해당 레시피에 노출

- **질문/답변**: "이런 결과 내려면 어떻게 해야 해요?" → 경험자 답변


## 핵심 지표 (KPI)

| 지표 | 측정 대상 | 목표 |
|------|----------|------|
| **레시피 완주율** | 레시피 시작 → 마지막 스텝 도달 비율 | 60% 이상 |
| **프롬프트 복사율** | 레시피 내 프롬프트 복사 클릭 수 | 조회 대비 40% |
| **재방문율** | 7일 내 재방문 비율 | 30% 이상 |
| **레시피 공유율** | 결과물 커뮤니티 공유 비율 | 레시피 완주자의 15% |
| **검색→레시피 전환율** | 검색 후 레시피 클릭 비율 | 25% 이상 |

## 5개 축 우선순위

1. **AI 레시피** (핵심 — 이것이 없으면 AIPICK은 또 하나의 AI 디렉토리)
2. **목적 기반 탐색** (유입 — 사용자가 찾아오는 이유)
3. **무료 사용량 정보** (차별화 — 가장 민감한 정보를 제공)
4. **스마트 추천** (리텐션 — 개인화로 재방문 유도)
5. **커뮤니티** (네트워크 효과 — 사용자가 콘텐츠를 만들어줌)
