# M3. 직군/학년별 추천 (Curation) 모듈

> 담당 경로: `/jobs`, `/jobs/[slug]`, `/education`, `/education/[level]`
> 관련 모듈: [M1-DISCOVERY](./M1-DISCOVERY.md) (위자드 Step 2), [M5-CONTENT](./M5-CONTENT.md) (가이드 연계)
> 최종 수정일: 2026-02-20

---

## 1. 모듈 개요

사용자의 "역할"(직군 또는 학년)에 맞춰 최적의 AI 툴킷을 제안하는 큐레이션 시스템.
단순 나열이 아니라 **추천 등급**(필수/추천/선택)과 **안전 등급**(안전/지도/고급)을 부여해
더 신뢰도 높은 추천을 제공한다.

---

## 2. 직군별 추천 (`/jobs`)

### 2.1 직군 목록 (10개)

| slug | 이름 | 핵심 AI 필요 영역 |
|------|------|----------------|
| ai-developer | AI 개발자 | 코딩, 모델 학습, 디버깅 |
| ux-designer | UI/UX 디자이너 | 와이어프레임, 프로토타입 |
| graphic-designer | 그래픽 디자이너 | 로고, 일러스트, 포스터 |
| marketer | 마케터 | 카피라이팅, SNS, 광고 |
| video-creator | 영상 크리에이터 | 유튜브, 숏폼, 편집 |
| writer | 작가/블로거 | 글쓰기, 블로그, 번역 |
| data-analyst | 데이터 분석가 | 시각화, 통계, 보고서 |
| entrepreneur | 사업가/창업자 | 사업 계획, 프레젠테이션 |
| musician | 음악가/작곡가 | 작곡, 편곡, 효과음 |
| pm | PM/기획자 | 기획서, 회의록, 일정 관리 |

> 상수 위치: `lib/constants.ts` → `JOB_CATEGORIES`

### 2.2 직군 목록 페이지 (`/jobs`)
- 10개 직군 카드 그리드 (아이콘 + 이름 + 간단 설명)
- "당신의 직업을 선택하세요" 헤드라인
- 정적 생성 (`generateStaticParams`)

### 2.3 직군 상세 페이지 (`/jobs/[slug]`)

```
┌─────────────────────────────────────────┐
│ 직군 소개 헤더 (이름, 설명, 아이콘)       │
├─────────────────────────────────────────┤
│ ★ 필수 AI 툴킷 (essential)              │
│   → 이 직군에 없어선 안 되는 도구 3~5개   │
├─────────────────────────────────────────┤
│ 추천 AI (recommended)                   │
│   → 적극 활용하면 좋은 도구              │
├─────────────────────────────────────────┤
│ 선택 AI (optional)                      │
│   → 상황에 따라 유용한 도구              │
├─────────────────────────────────────────┤
│ 관련 가이드 링크 (M5-CONTENT 연계)       │
└─────────────────────────────────────────┘
```

### 2.4 추천 등급 기준

| 등급 | 표시 | 기준 |
|------|------|------|
| essential | 🔴 필수 | 해당 직군 업무에 일상적으로 필수 |
| recommended | 🟡 추천 | 생산성을 크게 향상시키는 도구 |
| optional | 🟢 선택 | 특정 상황에서 유용 |

## 3. 학년별 추천 (`/education`)

### 3.1 학년 단계 (6단계)

| slug | 이름 | 대상 나이 | 핵심 초점 |
|------|------|---------|---------|
| elementary-lower | 초등 저학년 | 7~9세 | 안전한 AI, 학습 도우미, 창의력 |
| elementary-upper | 초등 고학년 | 10~12세 | 학습 보조, 코딩 입문, 영어 |
| middle | 중학생 | 13~15세 | 학습 보조, 영어, 코딩 입문 |
| high | 고등학생 | 16~18세 | 입시 도우미, 논술, 수학, 코딩 |
| college | 대학생/취준생 | 19~25세 | 논문, 리서치, 코딩, 데이터 분석 |
| teacher | 교사/교수 | — | 수업 설계, 과제 생성, 채점, 평가 |

> 상수 위치: `lib/constants.ts` → `EDU_LEVELS`

### 3.2 교육 메인 페이지 (`/education`)
- 6단계 학년 카드 그리드
- 각 카드에 대표 추천 AI 미리보기 (상위 2개)
- 안전 등급 범례 설명
- 정적 생성

### 3.3 학년 상세 페이지 (`/education/[level]`)

```
┌─────────────────────────────────────────┐
│ 학년 소개 (나이대, 설명)                  │
├─────────────────────────────────────────┤
│ 🟢 Safe — 자유롭게 사용 가능              │
│   → 연령에 완전히 적합한 AI              │
├─────────────────────────────────────────┤
│ 🟡 Guided — 보호자/교사 지도 하에 사용    │
│   → 올바른 활용법 교육이 필요한 AI       │
├─────────────────────────────────────────┤
│ 🔴 Advanced — 고학년/성인용              │
│   → 내용/기능상 어린 학생에게 부적합      │
├─────────────────────────────────────────┤
│ 학부모/교사 가이드 링크                   │
│ 관련 가이드 링크 (M5-CONTENT 연계)       │
└─────────────────────────────────────────┘
```

### 3.4 안전 등급 기준

| 등급 | 의미 | 표시 색상 |
|------|------|----------|
| safe | 해당 연령에 완전히 적합 | 🟢 Emerald |
| guided | 교육적 지도 하에 사용 권장 | 🟡 Amber |
| advanced | 성인/고급 사용자 대상 | 🔴 Red |

## 4. 데이터 구조

### 4.1 직군 추천 매핑

```
job_categories (직군 10개)
  └── job_tool_recommendations (직군-툴 N:M)
        ├── job_category_id → job_categories.id
        ├── tool_id → tools.id
        ├── recommendation_type (essential/recommended/optional)
        ├── reason (추천 이유 텍스트)
        └── sort_order
```

### 4.2 학년 추천 매핑

```
edu_levels (학년 6단계)
  └── edu_tool_recommendations (학년-툴 N:M)
        ├── edu_level_id → edu_levels.id
        ├── tool_id → tools.id
        ├── safety_rating (safe/guided/advanced)
        ├── use_case (활용 사례)
        └── sort_order
```

→ 전체 스키마: [DB-SCHEMA.md](../DB-SCHEMA.md)
