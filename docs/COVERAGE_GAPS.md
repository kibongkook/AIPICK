# AIPICK 커버리지 분석 및 누락 현황

## 🎯 현재 119개의 한계

### 1. 선정 편향 (Selection Bias)

#### 지리적 편향
- **과다 포함**: 미국 실리콘밸리 서비스 (70%)
- **과소 포함**:
  - 한국 AI 서비스 (3개만: 뤼튼, 뒤집기, 타입캐스트)
  - 중국 AI 서비스 (2개만: Kimi, 바이두)
  - 유럽 AI 서비스 (5개 미만)
  - 일본/동남아 AI 서비스 (거의 없음)

#### 규모 편향
- **과다 포함**: 대기업, 유니콘 스타트업
- **과소 포함**:
  - 초기 스타트업 (시드 단계)
  - 오픈소스 프로젝트
  - 개인 개발자 프로젝트
  - 무료/비영리 서비스

#### 카테고리 편향
- **과다 포함**: 글쓰기(30개), 영상(25개)
- **과소 포함**:
  - 학습·교육 (4개만)
  - 의료·헬스케어 (없음)
  - 법률·금융 (없음)
  - 부동산·건설 (없음)
  - 농업·환경 (없음)

### 2. 시간 지연 (Temporal Lag)

```
신규 출시 → AIPICK 등록
평균 지연: 2-6개월

이유:
- 수동 큐레이션 → 발견 지연
- 검증 과정 → 평가 기간 필요
- 안정성 확인 → 서비스 지속성 관찰
```

**결과**: 떠오르는 혁신적 서비스 누락 가능성

### 3. 접근성 편향

- **과다 포함**: 영어 서비스 (90%)
- **과소 포함**:
  - 한국어 전용 서비스
  - 로컬라이제이션 부족한 글로벌 서비스
  - 특정 국가만 접근 가능한 서비스

## 🔍 누락되었을 가능성 높은 서비스 유형

### 1. 전문가 도구 (Professional Tools)
```
예시:
- 의사용 의료 진단 AI
- 변호사용 판례 검색 AI
- 건축가용 설계 AI
- 금융 애널리스트용 예측 AI

누락 이유:
- B2B 전용, 일반 사용자 무관
- 높은 진입 장벽 (가격, 전문성)
- 제한된 접근성 (라이선스 필요)
```

### 2. 로컬 히어로 (Local Heroes)
```
예시:
- 한국 AI 스타트업 (200+개 추정)
- 지역 특화 번역기
- 한국어 맞춤법 검사기
- K-POP 음악 생성 AI

누락 이유:
- 글로벌 트래픽 데이터 부족
- Product Hunt 미등록
- 영어 자료 부족
```

### 3. 신생 혁신 (Fresh Innovation)
```
예시:
- 최근 2-3개월 내 출시
- Product Hunt 주간 1위
- HackerNews Show HN 폭발적 반응

누락 이유:
- 발견 지연 (자동화 전)
- 검증 기간 부족
- 안정성 미확인
```

### 4. 롱테일 니치 (Long-tail Niche)
```
예시:
- 특정 취미용 AI (낚시 장소 추천)
- 소수 언어 번역기 (웨일즈어, 바스크어)
- 초특화 분야 (AI로 악보 작곡하기)

누락 이유:
- 시장 규모 작음
- 우선순위 낮음
- 카테고리 분류 어려움
```

### 5. 오픈소스 자체 호스팅
```
예시:
- LocalLLaMA (로컬 실행)
- StableDiffusion (자체 설치)
- Whisper (로컬 음성 인식)

누락 이유:
- 웹 서비스 아님
- 방문수 측정 불가
- 기술적 진입 장벽
```

## 🎯 누락 서비스의 가치

### ❌ 소개할 의미 없는 경우:
```
1. 사기/스캠 서비스
2. 서비스 중단 예정
3. 심각한 보안/프라이버시 문제
4. 불법 콘텐츠 생성
5. 명백한 표절/복제품
```

### ✅ 소개할 의미 있지만 누락된 경우:
```
1. 신규 출시 (발견 지연)
2. 한국 로컬 서비스 (글로벌 데이터 부족)
3. 니치 분야 선두주자 (우선순위 문제)
4. 오픈소스 인기작 (웹 서비스 아님)
5. B2B 전문가 도구 (일반 사용자 접근성 낮음)

→ 이들은 **충분히 소개할 가치 있음**
```

## 📈 품질 vs 양의 균형

### 현재 철학: "큐레이션 > 종합 카탈로그"
```
✅ 장점:
- 신뢰도 높은 추천
- 선택 피로 감소
- 품질 보증

❌ 단점:
- 다양성 제한
- 신규 서비스 발견 지연
- 니치 분야 커버 부족
```

### 향후 방향: "스마트 필터링"
```
전체 DB: 1000개+
기본 노출: 상위 100개
고급 필터:
- "신규 서비스 보기"
- "니치 도구 보기"
- "한국 서비스만"
- "오픈소스만"
```

- ✅ 일반 사용자가 **지금 당장** 쓸 수 있는 검증된 도구들

**누락된 서비스 = 소개할 가치 없음 ❌**
**누락된 서비스 = 아직 발견 못함 또는 검증 중 ✅**
